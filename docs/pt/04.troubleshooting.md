# FAQ & Solução de Problemas

## Problemas Comuns

### O ícone da aba está girando mas nunca fica verde

Isso geralmente significa que o Syncthing ainda está escaneando ou o arquivo é grande. Verifique o painel web da GUI do Syncthing para progresso detalhado.

### O status do plugin mostra "Desconectado"

- Verifique se o serviço Syncthing está rodando.
- Certifique-se de que a Chave de API corresponde exatamente.
- No Android, verifique novamente se o HTTPS está desativado no App Syncthing.

### É seguro desativar o HTTPS no Android?

Sim. Ao definir o **Endereço de Escuta da GUI** para `127.0.0.1`, o acesso é restrito exclusivamente a aplicativos rodando localmente naquele dispositivo específico.
